# 物品移動システム

クライアントのみアプリケーション。データはローカルに保存します。

## 1. 物品・教室の登録
物品・教室を登録します

例）机、いす
例）1F 1-1, 2F 2-1
※教室は場所も登録します

## 2. 移動前個数の登録
移動前のそれぞれの教室にある個数を登録します

## 3. 移動後個数の登録
移動後の個数の登録をします

## 4. 移動計画の作成
物品移動は以下のルールに従って計画を製作します。

- まず同じフロア内での移動を優先します
- 可能であれば隣り合わせの教室での移動を優先
- 次に1F - 2F間、2F - 3F間のように上下フロアでの移動を優先
- 最後に余りの物品を移動させます
- それぞれの物品で余剰物品の置き場所を設定し、余剰物品はそこに移動します

## 5. 各教室用指示PDFの作成
どの物品をどこに移動するのかが記載されたPDFを教室分作成します
