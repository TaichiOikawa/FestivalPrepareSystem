# 実装タスク

## スコープ
- 仕様書の 1〜5 を実装する
- データのエクスポート/インポート（JSON）を実装する
- データはローカル保存（localStorage）する

## 実装順
1. 型定義と状態管理を実装する
2. 物品登録・教室登録・隣接教室設定を実装する
3. 移動前/移動後個数入力を実装する
4. 余剰物品置き場（物品ごとに1教室）を実装する
5. 移動計画エンジンを実装する（同フロア優先→隣接優先→上下フロア優先→余剰置き場）
6. 計画結果表示を実装する
7. PDF出力を実装する（全教室を1ファイル）
8. JSONエクスポート/インポートを実装する（schemaVersion付き）
9. 例外表示と入力バリデーションを実装する
10. lint/build を通す

## 仕様固定
- 隣接判定: 教室ごとに隣接教室リストを手入力
- 余剰置き場: 物品ごとに1教室
- PDF形式: 全教室1ファイル
- インポート挙動: 全置換のみ（マージなし）
